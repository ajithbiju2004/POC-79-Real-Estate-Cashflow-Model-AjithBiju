# POC 79 - Real Estate Cashflow Model

## Overview
A capital formation intelligence dashboard that models real estate cashflows, NOI, debt service and investor distributions under Bull, Base and Bear scenarios.

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- FastAPI
- Synthetic Real Estate Dataset

## Features
- Property Dashboard
- NOI Bridge
- Debt Service Metrics
- Scenario Analysis
- Cashflow Projection
- Cash Distribution Table
- Intelligence Panel
- Downloadable Dataset

## Data Sources
- Zillow Research
- FRED Economic Data
- Synthetic Property Portfolio Data

## Setup

Backend:

```bash
uvicorn main:app --reload

Frontend:

```bash
npm install
npm run dev
```

Application:

http://localhost:3000
